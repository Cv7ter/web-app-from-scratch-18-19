Doel van de web-app

De app heeft als doel om random foto‚Äôs te weergeven op de voorpagina. Er is een API gebruikt van unplash, een platform met fotoalbums erop die een API geven om de foto‚Äôs te gebruiken. Zo kan er op een foto van de pagina gedruk woren om nieuwe foto‚Äôs in te laden van dezelfde auteur. Met de link kan ik verschillende soorten foto‚Äôs krijgen, op de site worden random foto‚Äôs weergeven en als er dieper in de website gegaan wordt dan worden foto‚Äôs van de auteur weergeven op de nieuwe pagina.
De beperkingen bij deze API is dat er niet veel data in de API zit. Er kunnen foto‚Äôs ingeladen worden en in de API staat wat informatie over de auteur, titel en een beschrijving van de foto. Verder kan er geen informatie uit de API gehaald worden, waardoor het erg beperkt blijft om de link te gebruiken.
